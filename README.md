# webkit-sassui-toast
> Toast sass style.

## installation
```shell
npm i @feizheng/webkit-sassui-toast
```

## usage
+ https://afeiship.github.io/webkit-sassui-toast/

```scss
// in sass file:
@import 'node_modules/webkit-sassui-toast/dist/index.scss';

// webkit
@import '~@feizheng/webkit-sassui-toast';
```
